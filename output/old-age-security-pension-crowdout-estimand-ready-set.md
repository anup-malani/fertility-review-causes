# Estimand-ready set - old-age-security-pension-crowdout

The papers that identify the chapter's **primary estimand cell**: the old-age-security motive -> fertility (forward direction, fertility as the outcome). This is the set that can share a pooled estimate. It is the topical meta-analysis-ready set (§E1) intersected with the estimand-and-mechanism gate; off-cell papers are retained for their own cells, not here.

**10 of 40** reviewed studies are estimand-ready.
