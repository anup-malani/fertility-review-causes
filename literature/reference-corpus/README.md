# Reference Corpus for Meta-Analysis-to-Paper Pipeline

This folder contains local exemplar PDFs for designing the hybrid JEL-style / Cochrane-style
chapter pipeline. PDFs are intentionally grouped by the function they serve in the design work.

The PDFs are local reference material and are ignored by git via the repository-wide `*.pdf` rule.
This manifest is the durable inventory.

| File | Type | Why included | Notes |
|---|---|---|---|
| `jel-exemplars/fertility-framing/kearney-levine-2025-why-is-fertility-so-low-high-income-countries.pdf` | JEL fertility framing | Closest modern economics review to the low-fertility question. | Use for contemporary low-fertility framing, high-income-country narrative structure, and period/cohort distinctions. |
| `jel-exemplars/fertility-framing/guinnane-2011-historical-fertility-transition-guide-economists.pdf` | JEL fertility framing | Historical anchor for fertility-transition explanations. | Use for positioning the review against the economic-demographic transition literature. |
| `jel-exemplars/fertility-framing/greenwood-guner-vandenbroucke-2017-family-economics-writ-large.pdf` | JEL fertility framing | Broad family-economics synthesis spanning fertility, marriage, labor supply, and household structure. | Use for theory-section organization and cross-mechanism links. |
| `jel-exemplars/economics-review-style/keane-2011-labor-supply-and-taxes-survey.pdf` | JEL economics review style | Strong theory-to-evidence survey with behavioral margins and heterogeneous estimates. | Use for economic narrative style and elasticity/effect-size synthesis. |
| `jel-exemplars/economics-review-style/lusardi-mitchell-2014-financial-literacy-theory-evidence.pdf` | JEL economics review style | Clear movement from theory to measurement to evidence to policy relevance. | Use as a general model for integrating mechanisms, measurement problems, and empirical findings. |
| `economics-review-exemplars/abraham-kearney-2020-employment-population-ratio-review.pdf` | Quantitative economics review | Decomposes a major aggregate trend into candidate explanations. | Use for ranking mechanisms and separating quantified evidence from speculation. |
| `jel-exemplars/quantitative-meta-analysis/brown-imai-vieider-camerer-2024-loss-aversion-meta-analysis.pdf` | JEL quantitative meta-analysis | Explicit JEL meta-analysis with many estimates and study-design heterogeneity. | Use for coding estimates, pooled results, robustness checks, and quantitative evidence tables. |
| `jel-exemplars/quantitative-meta-analysis/stanley-2010-wheat-from-chaff-meta-analysis-quantitative-literature-review.pdf` | JEL quantitative meta-analysis | Methodological template for meta-analysis as quantitative literature review. | Use for meta-regression, publication-bias discussion, and economics-facing meta-analysis framing. |
| `jel-exemplars/quantitative-meta-analysis/jackson-mackevicius-2023-school-spending-policy-meta-analysis.pdf` | JEL quantitative meta-analysis | Policy meta-analysis with many quasi-experimental estimates. | Use for translating heterogeneous causal estimates into policy-relevant synthesis. |
| `jel-exemplars/quantitative-meta-analysis/parker-todd-2017-conditional-cash-transfers-progresa-oportunidades.pdf` | JEL/development evidence review | Program review combining theory, evaluation design, and empirical evidence. | Use for organizing mechanisms and evidence in a policy-relevant causal review. |
| `cochrane-exemplars/perinpanathan-2026-mobile-phone-contraception-use-cochrane.pdf` | Cochrane exemplar | Fertility-adjacent Cochrane review. | Use for PICO framing, risk-of-bias reporting, forest plots, and summary-of-findings conventions. |
| `cochrane-exemplars/oringanje-cd005215-adolescent-unintended-pregnancy-interventions-cochrane.pdf` | Cochrane exemplar | Broad fertility-related intervention synthesis with multiple intervention types. | Use for classifying interventions, handling multiple outcomes, and writing nuanced conclusions when evidence varies by outcome. |
| `cochrane-exemplars/lopez-cd007249-school-based-contraceptive-interventions-cochrane.pdf` | Cochrane exemplar | Example of a review where heterogeneity can prevent a simple pooled estimate. | Use for narrative synthesis, explaining why pooling is inappropriate, and organizing evidence by intervention type. |
| `cochrane-exemplars/lopez-cd011159-postpartum-contraceptive-use-strategies-cochrane.pdf` | Cochrane exemplar | Postpartum contraception review focused on fertility-adjacent program evidence. | Use for non-randomized evidence, risk-of-bias discussion, and cautious interpretation of program effects. |
| `cochrane-exemplars/lopez-cd012249-theory-based-contraception-interventions-cochrane.pdf` | Cochrane exemplar | Review organized around whether behavioral-theory-based interventions improve contraception. | Use for classifying studies by mechanism/theory/behavioral channel rather than only by outcome. |
